Guia Inicial de HTML

O que é HTML?

HTML é a linguagem de marcação usada para estruturar o conteúdo de uma página (texto, links, imagens, listas, tabelas, formulários etc.).

Observação: este guia foca em HTML “básico/tradicional”. Elementos semânticos do HTML5 (ex.: <header>, <nav>, <section>, <article>, <footer>, <main>, <video>, <audio>) ficarão para o outro documento.

Estrutura Básica de um Documento

<!DOCTYPE html>
<html>
  <head>
    <title>Título da Página</title>
  </head>
  <body>
    Conteúdo visível aqui.
  </body>
</html>

Tags principais

  • <!DOCTYPE html> — informa ao navegador o tipo de documento.
  • <html> — elemento raiz do documento.
  • <head> — configurações/metadata (não aparece no conteúdo).
  • <title> — título exibido na aba do navegador.
  • <body> — tudo o que aparece na página.

Formulários

Estrutura básica

<form action="enviar.php" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">

  <button type="submit">Enviar</button>
</form>

Inputs comuns

<input type="text" name="usuario">
<input type="password" name="senha">
<input type="email" name="email">
<input type="number" name="idade">
<input type="date" name="data">
<input type="checkbox" name="aceito">
<input type="radio" name="sexo" value="f">

Outros campos

<textarea name="mensagem" rows="4" cols="30"></textarea>

<select name="cidade">
  <option value="sp">São Paulo</option>
  <option value="rj">Rio de Janeiro</option>
</select>

Tags úteis em formulários

Tag Para que serve Exemplo
form Define o formulário <form action="x" method="post">
label Rótulo do campo <label for="email">Email</label>
input Campo de entrada <input type="text">
textarea Texto longo <textarea></textarea>
select Lista suspensa <select>...</select>
option Opção do select <option>...</option>
button Botão <button type="submit">
Dica prática: em formulários, o atributo name é o que “vai” no envio do formulário.

Atributos Comuns (muito usados)

Atributo Onde aparece Para que serve Exemplo
id quase qualquer tag Identificador único id="principal"
class quase qualquer tag Grupo/estilo via CSS class="destaque"
href a Destino do link href="pagina.html"
src img Caminho do arquivo src="img/foto.jpg"
alt img Texto alternativo alt="Logo"
title várias tags Dica ao passar o mouse title="Clique aqui"
name inputs Nome do dado enviado name="email"
value inputs Valor do campo value="123"
type input Tipo do input type="password"
placeholder inputs/textarea Texto de exemplo placeholder="Digite..."
required inputs Campo obrigatório required

Mini exemplo completo

<!DOCTYPE html>
<html>
  <head>
    <title>Exemplo</title>
  </head>
  <body>
    <h1>Cadastro</h1>

    <p>Preencha o formulário:</p>

    <form action="enviar.php" method="post">
      <label for="nome">Nome:</label>
      <input type="text" id="nome" name="nome" required>

      <br><br>

      <label for="email">Email:</label>
      <input type="email" id="email" name="email" placeholder="ex: ana@site.com">

      <br><br>

      <button type="submit">Enviar</button>
    </form>

    <hr>

    <p>Acesse nosso <a href="https://exemplo.com">site</a>.</p>
  </body>
</html>
