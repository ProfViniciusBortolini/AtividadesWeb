Guia Inicial de HTML

Estrutura Básica

<!DOCTYPE html>
<html>
  <head>
    <title>Título da Página</title>
  </head>
  <body>
    Conteúdo visível aqui.
  </body>
</html>
Resultado:
Conteúdo visível aqui.

Títulos

<h3>Este é um h3</h3>
<h4>Este é um h4</h4>
<h5>Este é um h5</h5>
<h6>Este é um h6</h6>

Este é um h2

Este é um h3

Este é um h4

Listas

Lista não ordenada

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
Resultado:
  • Item 1
  • Item 2

Lista ordenada

<ol>
  <li>Passo 1</li>
  <li>Passo 2</li>
</ol>
Resultado:
  1. Passo 1
  2. Passo 2

Tabelas

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>22</td>
  </tr>
</table>
Resultado:
Nome Idade
Ana 22
